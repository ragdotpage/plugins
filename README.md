# yboard-dev

Claude Code plugin for building and deploying full web applications on the Yboard platform.

## Installation

```bash
claude plugin add /path/to/yboard-dev
```

## Usage

Just describe what you want to build. The skill triggers on requests like:
- "Make me an app"
- "Build a website for my business"
- "I want to create a dashboard"
- "Deploy a SaaS tool"

The agent handles everything end-to-end — authentication, setup, coding, building, and deployment.
